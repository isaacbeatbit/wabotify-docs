# Archivos

Agregar archivos a un Agente de IA

## API de Archivos – Agregar, Validar y Eliminar Archivos

El agente tiene la capacidad de poder leer archivos que el usuario pueda subir, estos archivos pueden ser de tipo PDF, DOC y DOCX.

![Crear Nuevo Agente](../../../assets/agente-ia/files.avif)

### Campos del Archivo

| Campo     | Descripción                                                                                                                                                                            |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| fileUrl   | URL pública del archivo a subir. Solo se permiten los formatos .pdf, .doc, .docx.                                                                                                      |
| reference | Descripción detallada del contenido del archivo. Se utiliza para poder darle información al agente y sepa de donde sacar la información que consulta el usuario.                       |
| tags      | Son referencias para poder entender más la información dentro del archivo. Si bien es cierto, con la Referencia existe más posibilidades, con los tags complementa esta funcionalidad. |
