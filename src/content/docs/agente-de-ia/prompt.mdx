# Prompt

Definición de como funcionan los prompts del Agente de IA

## Introducción

Este documento menciona como funcionan el prompt para el Agente de IA.

El prompt son las instrucciones que el Agente tendrá de para poder contestar al usuario que realiza una consulta.

![Crear Nuevo Agente](../../../assets/agente-ia/prompt.avif)

## Apoyo de la IA

Cuando se crea un prompt, la plataforma te da la opción de poder usar la IA para mejorar los discursos y parrafos de una forma muy sencilla. Solo se tiene que sombrear el parrafo a mejorar y saldrá la opción para mejorar el texto.

![Crear Nuevo Agente](../../../assets/agente-ia/prompt-2.avif)

Luego saldrá la opción para mejorar el texto y podrá Aceptar o Rechazar el cambio.

![Crear Nuevo Agente](../../../assets/agente-ia/prompt-3.avif)
