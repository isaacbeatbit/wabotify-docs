## Formato de Fecha (`date`)

Los formatos de fecha se definen con patrones tipo `yyyy-MM-dd` o abreviaciones.

| Formato        | Resultado para `2025-06-01` |
| -------------- | --------------------------- |
| `yyyy-MM-dd`   | `2025-06-01`                |
| `dd/MM/yyyy`   | `01/06/2025`                |
| `MMM dd, yyyy` | `Jun 01, 2025`              |
| `EEEE dd MMMM` | `Sunday 01 June`            |
| `d 'de' MMMM`  | `1 de junio`                |

**Ejemplos**:

- `{{ fechaNacimiento | format:'dd/MM/yyyy' }}`

**Ejemplos**:
